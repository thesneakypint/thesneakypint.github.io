    Draw a line on 32x32 display. Emulator provides a 32x32 pixel display. Display memory starts at $0200.

    
LDA #$02
STA $00 ; display address MSB

LDA #$00
STA $01 ; display address LSB

LDA #$01 ; color = 1 (white)

drawline
STA ($00),Y
INY
CPY #$20
BNE drawline
